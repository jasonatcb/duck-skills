# Screenshot Highlight Skill

為截圖加上燈箱效果（Lightbox dim effect），僅保留指定區域的亮度，其他區域變暗以突顯重點。

## 使用方式

**觸發時機**：用戶說「加入螢光筆」、「標記重點」、「highlight」、「燈箱效果」、「dim effect」、「畫面標註」或提供圖片並說要標記某個元素。

**輸入**：
1. 圖片檔案路徑
2. 要突顯的元素（從選項中選擇）

**輸出**：加上燈箱效果的圖片，儲存至原圖片同目錄，檔名加上 `_highlighted` 後綴。

## 執行步驟

### 1. 讀取圖片

使用 Read 工具讀取圖片，分析圖片內容並識別可標記的元素。

### 2. 列出可標記的選項

根據圖片內容，常見的元素類型包括：

| 圖片類型 | 常見可標記元素 |
|----------|----------------|
| 表格 | 欄位名稱（日期、狀態、金額）、表格列、欄位標題 |
| 表單 | 輸入欄位、按鈕、核取方塊、下拉選單 |
| 導覽 | 選單項目、分頁籤、側邊欄項目 |
| 列表 | 列表項目、勾選框、動作按鈕 |
| 訊息 | 狀態訊息、錯誤提示、成功訊息 |

### 3. 詢問用戶

向用戶列出發現的元素選項，詢問要標記哪個：

```
圖片中發現以下可標記的元素：
1. 欄位名稱（例：廣告名稱、狀態、建立時間）
2. 資料列（例：某一列資料）
3. 按鈕（例：編輯、刪除、暫停）
4. 狀態標籤（例：投放中、已暫停）

請問要標記哪個元素？(可以說編號或描述)
```

### 4. 取得座標

根據選擇的元素，估算其在圖片中的位置和尺寸：

- **欄位名稱**：通常在表格頂部，寬度約為欄位寬度，高度約 20-30px
- **資料列**：根據列索引計算 Y 座標
- **按鈕**：通常在列的右側，寬度約 60-80px，高度約 30px
- **狀態標籤**：根據狀態文字位置，寬度約 60-80px

### 5. 繪製燈箱效果

使用 Python + PIL/Pillow 在圖片上產生燈箱效果（僅暗化，非目標區域變暗 55%）：

```python
from PIL import Image, ImageDraw

def create_lightbox_effect(image_path, output_path, x, y, width, height, dim_opacity=0.55):
    """
    創建燈箱效果 - 僅暗化非目標區域
    
    參數：
    - x, y, width, height: 要突顯的區域座標
    - dim_opacity: 變暗程度 (0-1)，預設 0.55
    """
    img = Image.open(image_path)
    img = img.convert("RGBA")
    
    # 建立暗色遮罩
    overlay = Image.new("RGBA", img.size, (0, 0, 0, 0))
    draw = ImageDraw.Draw(overlay)
    
    # 填充整個圖片為暗色
    alpha = int(255 * dim_opacity)
    draw.rectangle([(0, 0), img.size], fill=(0, 0, 0, alpha))
    
    # 在目標區域挖洞（保持明亮）
    draw.rectangle([(x, y), (x + width, y + height)], fill=(0, 0, 0, 0))
    
    # 組合原圖和遮罩
    result = Image.alpha_composite(img, overlay)
    result = result.convert("RGB")
    result.save(output_path)
```

### 6. 儲存結果

輸出檔案命名規則：`{原始檔名}_highlighted.{副檔名}`

例如：
- `ec-maa-廣告活動列表-廣告名稱.png` → `ec-maa-廣告活動列表-廣告名稱_highlighted.png`

## 燈箱效果參數

| 參數 | 說明 | 預設值 |
|------|------|--------|
| dim_opacity | 變暗程度 (0-1) | 0.55 (約 55%，根據參考圖片) |

## 常見圖片類型的標記位置參考

### 表格類型
- **欄位名稱**：y = 50-80（左對齊）
- **第一列資料**：y = 80-110
- **狀態欄**：根據圖片寬度約 70-80% 位置

### 按鈕類型
- **操作按鈕**：圖片右側，寬度約 60-80px

### 狀態標籤
- **投放中/已暫停**：表格內，寬度約 60-80px

## 錯誤處理

- 如果無法識別元素，回覆「無法自動識別，請提供要標記的區域座標」
- 如果圖片格式不支援，提示用戶支援的格式（PNG、JPG、JPEG）

**輸入**：
```
@docs/assets/images/ec-maa-廣告活動列表-廣告名稱.png
```

**輸出**：
