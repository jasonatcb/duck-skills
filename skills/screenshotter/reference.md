# 截圖擺放位置參考指南

本文檔說明截圖在技術文檔中的正確擺放位置，確保 Markdown 渲染正確。

## 1. Hero 頁面截圖

**位置：** 文件標題說明後的第一個區塊（section）開頭

**用途：** 展示目標頁面/功能的整體外觀

**格式：**
```markdown
# 文件標題
說明文字
{ .subtitle }

![描述文字](../../../assets/images/圖檔名.png){ .hero-page }

## 第一個區塊標題
```

**範例：**
```markdown
# 設定 Google Analytics

說明文字
{ .subtitle }

![GA4 設定頁面](../../../assets/images/ec-ga4-設定.png){ .hero-page }

## 開始前的準備
```

## 2. 步驟清單中的截圖

### 2.1 截圖位於步驟之間

**位置：** 在相關步驟之後，下一個步驟之前

**縮排：** 4 空格（與子項目同级）

**格式：**
```markdown
1.  **第一步**：描述動作。
2.  **第二步**：描述動作。

    ![截圖描述](../../../assets/images/圖檔名.png){ .screenshot }

3.  **第三步**：描述動作。
```

**範例：**
```markdown
1.  **進入轉換頁面**：於側邊欄點擊「目標」，點擊「建立轉換動作」。
2.  **選擇來源**：選擇「網站」。

    ![Google Ads 轉換動作頁面](../../../assets/images/google-ads-轉換動作頁面.png){ .screenshot }

3.  **掃描網址**：輸入官網網址，點擊「掃描」。
```

### 2.2 截圖位於步驟結束後

**位置：** 一系列步驟完成後，下一個區塊之前

**用途：** 展示設定完成後的結果畫面

**格式：**
```markdown
1.  **步驟一**：描述動作。
2.  **步驟二**：描述動作。
3.  **步驟三**：描述動作。

    ![截圖描述](../../../assets/images/圖檔名.png){ .screenshot }

## 下一個區塊
```

**範例：**
```markdown
1.  **Google Ads 再行銷**：填入轉換 ID。
2.  **Google Ads 轉換追蹤**：設定轉換事件。
3.  **儲存設定**：點擊儲存按鈕。

    ![谷歌 Google 設定頁面](../../../assets/images/ec-谷歌-設定.png){ .screenshot }

## 重要注意事項
```

## 3. 路徑說明 + 截圖

**位置：** 路徑說明文字之後，若有後續步驟，截圖應放在步驟之後

**錯誤擺放：**
```markdown
**路徑：** 「第三方整合」 > 「谷歌 Google 設定」。

    ![截圖]  <- 錯誤：放在步驟之前

1.  第一步...
```

**正確擺放：**
```markdown
**路徑：** 「第三方整合」 > 「谷歌 Google 設定」。

1.  第一步...
2.  第二步...

    ![截圖]  <- 正確：放在步驟之後
```

## 4. 子項目中的截圖

**位置：** 子項目（`*` 或 `-` 開頭的項目）內部或之後

**縮排：** 4 空格（與子項目內容同級）

**格式：**
```markdown
*   **標題**：描述
    *   子項目描述

        ![截圖描述](../../../assets/images/圖檔名.png){ .screenshot }
```

**範例：**
```markdown
*   **設定路徑**：在 GA4 後台點擊「管理」>「資料收集」>「開啟加強型評估」。

    ![開啟加強型評估.gif](../../../assets/images/ec-ga4-開啟加強型評估.gif)
```

---

## 5. Grid Cards 中的截圖

**位置：** `<div class="grid cards">` 區塊內

**用途：** 並排展示多個畫面或功能

**格式：**
```markdown
<div class="grid cards" markdown>

- :icon: 標題
  說明
  ![截圖描述](../../../assets/images/圖檔名.png){ .screenshot }

</div>
```

---

## 6. 一般說明文字後的截圖

**位置：** 段落說明文字與下一個區塊標題之間

**用途：** 補充說明或展示結果

**格式：**
```markdown
說明文字...

![截圖描述](../../../assets/images/圖檔名.png){ .screenshot }

## 下一個區塊
```

## 快速對照表

| 場景 | 截圖位置 | 縮排 |
|-----|---------|------|
| Hero 頁面 | 文件開頭（第一個區塊前） | 無 |
| 步驟之間 | 步驟 N 之後、步驟 N+1 之前 | 4 空格 |
| 步驟結果 | 最後一個步驟之後 | 4 空格 |
| 路徑說明 | 步驟清單完成之後（非路徑與步驟之間） | 4 空格 |
| 子項目 | 子項目內容之後 | 4 空格 |
| Grid Cards | 卡片內容區塊內 | 無 |
