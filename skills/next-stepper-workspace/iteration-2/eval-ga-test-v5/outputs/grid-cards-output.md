:material-filter-outline: **排除內部流量** { .tw-text-blue-600 }

在 GA4 管理介面的「資料串流」中定義公司 IP，避免開發或行銷人員的瀏覽行為干擾分析。

---

:material-link-variant-remove: **列出不適用的參照連結** { .tw-text-purple-600 }

將金物流服務商（如 `cyberbizpay.com`、`pay.ecpay.com.tw`、`web-pay.line.me` 等）加入排除名單，以免轉換來源被誤判為第三方金流頁面。

---

:material-clock-outline: **延長資料保留期限** { .tw-text-green-600 }

GA4 預設資料僅保留 2 個月，建議至「資料收集與修改」>「資料保留」中手動改為 **14 個月**。

---

:material-account-outline: **啟用 Google 信號** { .tw-text-orange-600 }

開啟此功能可取得跨裝置的使用行為資料與更精確的使用者輪廓。
